🧠 Smart Data Analyst Dashboard

🚀 Overview

The Smart Data Analyst Dashboard is an AI-powered web application built using Streamlit that allows users to upload CSV/Excel files and instantly generate insights, visualizations, and AI-powered analysis using LLMs (Groq API).

It helps non-technical users understand data easily through charts, summaries, and intelligent recommendations.

✨ Features

📊 Interactive Dashboard

Dataset preview
Rows, columns, missing values, duplicates
Statistical summary

📈 Data Visualization

-Histogram
-Bar charts
-Pie charts
-Interactive Plotly graphs

🧠 AI Insights (LLM Powered)

-Data summary
-Trends detection
-Business insights
-Recommendations

🔍 Data Quality Report

-Missing values analysis
-Duplicate detection
-Data health score
-Column-wise issue report

💬 AI Analysis Chat

-Ask questions about dataset
-AI-generated answers using Groq LLM

📥 Export Feature

-Download dataset
-Download AI insights report

🛠️ Tech Stack
-Python 
-Streamlit 
-Pandas 
-Plotly 
-Groq API 
-LLM: LLaMA 3.3 
-dotenv
